Что исследовали: как предсказать медицинские расходы человека по его возрасту, весу, курению и другим признакам.

Построение и сравнение регрессионных моделей для прогнозирования медицинских расходов.


Источник: insurance.csv (Medical Cost Personal Datasets, Kaggle)
Размер: 1338 записей, 7 признаков

Как исследовали: 
Загрузили данные (1338 человек, 7 признаков)
Провели разведочный анализ — графики, корреляции, выбросы
Закодировали категории, создали новые признаки (Feature Engineering)
Обучили 6 моделей: Lasso, Random Forest, 4 версии CatBoost
Подобрали гиперпараметры через RandomizedSearchCV



Результат исследования:
Обучены 6 моделей: Lasso Regression, Random Forest и 4 версии CatBoost
каждая модель оценена по трём метрикам: MAE, RMSE, R²

Ключевые выводы: 
Лучшая модель по MAE: CatBoost v3 (Feature Engineering) — средняя ошибка 2283 у.е.
Лучшая модель по R²: Random Forest — объясняет 89.6% дисперсии данных.
Главный фактор расходов: курение (влияние ~70-76% по важности признаков).
Количество детей слабо влияет на расходы.
Взаимодействие курения и ожирения: курящие с ожирением имеют расходы в 2-3 раза выше, чем курящие с нормальным весом.
